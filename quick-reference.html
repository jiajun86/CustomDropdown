Defining Dropdown

Via JavaScript

Call a Custom Dropdown with id myDropdown with a single line of JavaScript: 

$('#myDropdown').modal(options);

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-dropdown-, as in data-dropdown-allowEmpty="". For JavaScript, append the option name to dropdown. as in dropdown.allowEmpty="".

Name type default description
allowEmpty boolean true Allow empty option to be selected
autoClose boolean true Automatically closes the dialog box once any option selected
capitalize boolean true Capitalizes display text for all options
showDialogBox boolean true Uses dialog box for displaying options
showPagination boolean true Shows pagination bar for page navigation
showSearchBar boolean true Shows search bar for option filtering
showSpacer boolean true Uses spacer to fill up empty spaces for consistency among all pages
maxPerPage integer 24 Maximum number of options shown per page
maxPerColumn integer 12 Maximum number of columns shown per page
maxPerPagination integer 10 Maximum number of pages shown per pagination
options string '' JSON string representing an array of JavaScript objects to use as options for selection
textKey string 'text' JavaScript object property name to use as option text
valueKey string 'value' JavaScript object property name to use as option value
sortBy string 'text' JavaScript object property name for sorting the options

Customizing Buttons

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-button-, as in data-button-rowClass="". For JavaScript, append the option name to button. as in button.rowClass="".

Name type default description
rowClass string 'row' Uses row or row-fluid to change between fixed and fluid grid system
defaultClass string 'btn btn-block' CSS class to use as default state of buttons. Accepts Bootstrap button styles and sizes
activeClass string 'btn-info' CSS class to use as active state of buttons. Accepts Bootstrap button styles and sizes
inactiveClass string '' CSS class to use as inactive state of buttons. Accepts Bootstrap button styles and sizes
defaultIconClass string '' CSS class to use as default state of icons. Accepts Font Awesome icons
activeIconClass string 'icon-check' CSS class to use as active state of icons. Accepts Font Awesome icons
inactiveIconClass string 'icon-check-empty' CSS class to use as inactive state of icons. Accepts Font Awesome icons
fadeOut integer 300 Defines the speed of buttons fading out when changing page

Controlling Pagination

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-pagination-, as in data-pagination-wrapperClass="". For JavaScript, append the option name to pagination. as in pagination.wrapperClass="".

Name type default description
wrapperClass string 'pagination' CSS class to use as wrapper for pagination. Accepts Bootstrap pagination styles, sizes and alignment
defaultClass string '' CSS class to use as default state of pagination buttons. Accepts Bootstrap button styles
activeClass string 'btn-info' CSS class to use as active state of pagination buttons. Accepts Bootstrap button styles
inactiveClass string '' CSS class to use as inactive state of pagination buttons. Accepts Bootstrap button styles
disabledClass string 'active' CSS class to use as disabled state of pagination buttons. Accepts Bootstrap button styles
previousClass string 'icon-double-angle-left' CSS class to use as previous icon. Accepts Font Awesome icons
nextClass string 'icon-double-angle-right' CSS class to use as next icon. Accepts Font Awesome icons
fadeOut integer 0 Defines the speed of pagination fading out when changing pagination page
