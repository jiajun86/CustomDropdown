General

Example

Dropdown as toggleable buttons. 

<select class="dropdown-example"
        data-dropdown-allowEmpty="false"
        data-dropdown-showDialogBox="false"
        data-dropdown-showPagination="false"
        data-dropdown-showSearchBar="false"
        data-dropdown-maxPerPage="6"
        data-dropdown-maxPerColumn="3"
        data-button-rowClass="row-fluid">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
  <option value="4">Option 4</option>
  <option value="5">Option 5</option>
  <option value="6">Option 6</option>
</select>

  <option value="2">Option 2</option>

Dialog Box

Enable dialog box to display toggleable buttons separately in a dialog box. 

<select class="dropdown-example"
        data-dropdown-allowEmpty="false"
        data-dropdown-showDialogBox="true"
        data-dropdown-showPagination="false"
        data-dropdown-showSearchBar="false"
        data-dropdown-maxPerPage="6"
        data-dropdown-maxPerColumn="3"
        data-button-rowClass="row-fluid">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
  <option value="4">Option 4</option>
  <option value="5">Option 5</option>
  <option value="6">Option 6</option>
</select>

Pagination

Enable pagination if total number of options exceed the number of buttons supported per page. 

<select class="dropdown-example"
        data-dropdown-allowEmpty="false"
        data-dropdown-showDialogBox="false"
        data-dropdown-showPagination="true"
        data-dropdown-showSearchBar="false"
        data-dropdown-maxPerPage="3"
        data-dropdown-maxPerColumn="1"
        data-button-rowClass="row-fluid">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
  <option value="4">Option 4</option>
  <option value="5">Option 5</option>
  <option value="6">Option 6</option>
  <option value="7">Option 7</option>
  <option value="8">Option 8</option>
  <option value="9">Option 9</option>
</select>

Single Column

Example

Dropdown as a single column of toggleable buttons by setting data-dropdown-maxPerColumn="1". Supports up to maximum of 24 buttons. Enable pagination bar if more than 24 buttons by setting data-dropdown-showPagination="true". 

<select class="dropdown-example"
        data-dropdown-allowEmpty="false"
        data-dropdown-showDialogBox="false"
        data-dropdown-showPagination="false"
        data-dropdown-showSearchBar="false"
        data-dropdown-maxPerColumn="1"
        data-button-rowClass="row-fluid">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
</select>

Button Width

Control button width by wrapping it in a <div>

<div style="width: 200px">
  <select class="dropdown-example"
          data-dropdown-allowEmpty="false"
          data-dropdown-showDialogBox="false"
          data-dropdown-showPagination="false"
          data-dropdown-showSearchBar="false"
          data-dropdown-maxPerColumn="1"
          data-button-rowClass="row-fluid">
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    <option value="3">Option 3</option>
  </select>
</div>

Single Row

Example

Dropdown as a single row of toggleable buttons by setting data-dropdown-maxPerPage the same as data-dropdown-maxPerColumn. Supports up to maximum of 12 buttons. Enable pagination bar if more than 12 buttons by setting data-dropdown-showPagination="true". 

<select class="dropdown-example"
        data-dropdown-allowEmpty="false"
        data-dropdown-showDialogBox="false"
        data-dropdown-showPagination="false"
        data-dropdown-showSearchBar="false"
        data-dropdown-maxPerPage="3"
        data-dropdown-maxPerColumn="3"
        data-button-rowClass="row-fluid">
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
  <option value="3">Option 3</option>
</select>
